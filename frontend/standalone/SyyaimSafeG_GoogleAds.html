Campaign Setup & Bidding Tips

🎯
Bidding Strategy
Start with Maximize Clicks (budget ₹500–800/day) for 2 weeks to gather data, then switch to Target CPA once you have 30+ conversions.
📍
Geo Targeting
Start with Maharashtra, Gujarat, Haryana, Tamil Nadu, Karnataka — highest density of registered factories. Expand to UP & Rajasthan in Phase 2.
🕐
Ad Scheduling
Set higher bids Mon–Fri 9AM–6PM IST (when HSE managers are at desks). Reduce bids 40% on weekends. Use bid adjustments by device — desktop +20%.
🔁
Audience Layering
Add In-Market audiences: "Business Services > EHS Software", "Manufacturing > Industrial Equipment". Use Customer Match to upload existing lead list.
📊
Conversion Tracking
Track: Demo form submit (primary), Pricing page visit (secondary), Video watched 30s+. Set conversion value = ₹50,000 (avg pilot deal value) for smart bidding.
🚫
Negative Keywords
Add: free, diy, training course, certification exam, jobs, career, salary, pdf, ppt, wikipedia, meaning, definition. Protects budget from informational queries.
🔗
Landing Page Match
Each ad variant needs its own landing page. Variant A → PPE detection demo. Variant B → ROI calculator. Variant C → Compliance checklist. Match headline to LP hero text exactly.
📈
Estimated Performance
Expected CPC: ₹80–180 for safety software keywords. CTR target: 6–10% (high-intent B2B). Budget: ₹25,000–40,000/month to generate 15–20 demo bookings.
